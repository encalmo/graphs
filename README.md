# graphs

Scala library for processing graphs.

## Dependencies

   - JVM >= 21
   - [Scala](https://www.scala-lang.org/) >= 3.5.2

## Usage

Use with SBT

    libraryDependencies += "org.encalmo" % "graphs_3" % "0.9.3"

or with SCALA-CLI

    //> using dep org.encalmo::graphs:0.9.3
